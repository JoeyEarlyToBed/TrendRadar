🔥 NewsNow 自部署指南

TrendRadar 自定义数据源部署方案

⚠️ 当前状态

NewsNow 官方 API (https://newsnow.busiyi.world/api/s) 已被 Cloudflare 拦截，
直接访问返回 403 错误。需要自部署才能使用。

🚀 推荐部署方案

  • 1
    Fork NewsNow 仓库
    访问 https://github.com/ourongxing/newsnow
    点击右上角 "Fork" 按钮，等待 Fork 完成
  • 2
    部署到 Cloudflare Pages（推荐）
    1. 登录 https://dash.cloudflare.com（免费注册）
    2. 点击 "Pages" → "Create a project"
    3. 选择 "Connect to Git"，授权 GitHub
    4. 选择你 Fork 的 newsnow 仓库
    5. 构建设置：
       - Build command: npm install -g pnpm && pnpm install && pnpm run build
       - Output directory: dist/output/public
    6. 点击 "Save and Deploy"，等待 2-3 分钟
  • 3
    配置 TrendRadar
    编辑 config/config.yaml：
    api_url: "https://你的域名/api/s"

    例如：api_url: "https://newsnow-xxx.pages.dev/api/s"

📊 部署后效果

• 可视化热榜页面：https://你的域名/
• API 端点（TrendRadar 使用）：https://你的域名/api/s
• 支持 23+ 平台实时热榜
• 30 分钟缓存，自动更新

🔧 备选方案

方案 A：Vercel 部署
类似 Cloudflare，访问 vercel.com，导入 GitHub 仓库即可

方案 B：Docker 本地部署
需要 Docker 环境：
git clone https://github.com/ourongxing/newsnow.git
cd newsnow && docker compose up

方案 C：Node.js 本地部署
需要 Node.js + pnpm：
pnpm install && pnpm run dev
