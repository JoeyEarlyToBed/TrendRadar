Architecture

核心架构

TrendRadar 基于 Python 构建，从多层数据源采集 → 关键词引擎过滤 → AI 深度分析 → 多渠道推送，形成完整的情报处理流水线。

📡
数据采集层
  • NewsNow API — 11 个热榜平台
  • RSS 2.0 / Atom / JSON Feed
  • 自动重试 + 请求间隔控制
  • 域名安全校验
🎯
匹配过滤层
  • 全局过滤器排除标题党
  • 正则 / +必须词 / !排除词
  • 词组别名 & 条数限制
  • 增量模式过滤已读
🧠
AI 分析层
  • DeepSeek / OpenAI 兼容 API
  • 6 板块结构化分析报告
  • 跨平台关联分析
  • 弱信号 & 轨迹突变捕捉
📊
报告生成层
  • HTML 可视化报告
